🎮 게임 방법

클릭/터치하면 너구리가 방귀를 뀌며 위로 올라갑니다 💨
떼면 중력에 의해 아래로 내려갑니다
동굴 위아래 벽에 닿으면 게임 오버
오래 살아남을수록 동굴이 좁아지고 속도가 빨라집니다


💨 조작법
환경조작PC마우스 클릭 누르고 있기모바일화면 터치하고 있기

⚙️ 난이도 선택
시작 전 속도를 선택할 수 있어요:
난이도설명🐢 하느린 속도로 시작, 천천히 빨라짐🦝 중기본 속도 (추천)🔥 상빠른 속도로 시작, 빠르게 빨라짐

✨ 특징

💨 방귀 사운드 — 클릭/터치할 때마다 실감나는 방귀 효과음
🦝 너구리 캐릭터 — 줄무늬 꼬리, 눈 마스크, 귀여운 외모
🌀 동굴 변화 — 위아래로 구불구불, 좁아졌다 넓어졌다 자연스럽게 변화
📈 점점 빨라지는 속도 — 시간이 갈수록 스크롤 속도 증가
🏆 최고 기록 저장 — 브라우저에 자동 저장
